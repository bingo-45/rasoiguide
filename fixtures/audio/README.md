# Audio fixtures

These deterministic 44.1 kHz mono WAV files exercise the whistle detector without licensing ambiguity:

- `single-whistle.wav`: one 1.15 second pressure-cooker-like harmonic burst.
- `three-whistles.wav`: three bursts separated by more than the eight-second refractory window.
- `exhaust-fan-only.wav`: low-frequency fan hum plus broadband room noise.
- `mixer-grinder-only.wav`: a sustained motor spectrum that must be rejected by the maximum-duration gate.

Run `pnpm fixtures:audio` to reproduce them byte-for-byte. They are engineered fixtures, not substitutes for the documented physical-kitchen device pass.

The app bell is an original, procedurally struck four-partial brass timbre generated by the same script. It contains no third-party recording.
